# EDS 232 – Labs

Weekly lab assignments for **EDS 232: Machine Learning for Environmental Science** at UC Santa Barbara's Bren School of Environmental Science & Management.

## Contents

Completed lab notebooks applying ML methods to environmental datasets. Each lab folder contains the assignment notebook and any associated data or outputs.

## Setup

This repo uses the `eds232-env` conda environment. To build it:
```bash
conda env create --name eds232-env --file eds232-env.yml
conda activate eds232-env
```

Then register the Jupyter kernel:
```bash
python -m ipykernel install --user --name eds232-env --display-name "eds232-env"
```

See the [course setup page](https://meds-eds-232.github.io/EDS-232-course-website/notes/setup/setup.html) for full instructions.
